1、找 Hexo 主题，丢上 Github 仓库 1。
2.1（二选一之一）、生成 RSA 公私钥对，私钥放仓库 1 Secrets。新建仓库 2，命名为用户名.github.io。公钥放仓库 2 Deploy Key。
2.2、配置 config.yml，使用现成 Github Action“Hexo Deploy”。
2.3、完成。

---

3.1（二选一之二）、使用 netlify，使用 github 账户登录，导入仓库 1.
3.2、完成。
